# 工银智慧温暖家 🏠

一个基于 React + Vite 的移动端优先的温暖家居应用。

## ✨ 特性

- 🎨 多主题切换（粉色、黄色、绿色、蓝色）
- 📱 移动端优先设计，支持全面屏适配
- 🚀 基于 Vite 6.x，快速构建和热更新
- 💅 Tailwind CSS 4.x，现代化样式方案
- 📊 数据可视化图表（Recharts）
- 🎭 流畅的动画效果（Motion）

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 📦 部署

### GitHub Pages 部署

1. 修改 `vite.config.ts` 中的 `base` 为你的仓库名
2. 推送到 GitHub
3. 启用 GitHub Actions
4. 自动部署到 `https://your-username.github.io/icbc-smart-warm-home/`

详见 [DEPLOYMENT.md](./DEPLOYMENT.md)

## 🛠️ 技术栈

- **框架**: React 19
- **构建工具**: Vite 6
- **样式**: Tailwind CSS 4
- **动画**: Motion
- **图表**: Recharts
- **图标**: Lucide React
